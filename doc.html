Dokumentacja

Projekt został wykonany z wykorzystaniem technologii HTML5, CSS oraz JavaScript.

Animacja została wykonana w technologii Canvas.

Elementy technologii HTML5:

  • elementy semantyczne w tym: <header> , <nav> , <footer>
  • Grafika z wykorzystaniem elementu <canvas>

Elementy technologii CSS:

  • stylowanie aplikacji za pomocą atrybutów class elementów <div>
  • użycie selektorów CSS w tym first-of-type, last-of-type, focus, active, first-child
  • użycie własności 'transform', 'text-transform'

Elementy technologii JavaScript:

  • modyfikacje dokumentu HTML przy użyciu funkcji 'innerHtml', 'getElementById', 'querySelector', 'insertAdjacentHtml', 'removeChild'
  • obsługa zdarzeń za pomocą addEventListener, obsługiwane zdarzenia to m.in. 'click', 'mousedown', 'mousemove', 'mouseover', 'keypress'
  • obsługa elementu <canvas> za pomocą funkcji 'getContext', 'beginPath', 'arc', 'fill', 'fillRect', 'getImageData', 'putImageData', 'moveTo', 'lineTo', 'stroke'
  • realizacja pliku js w oparciu o wzorzec MVC, każda część została wykonana jako IIFE
